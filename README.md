# Frontend Assessment Project

## Overview

This project is a responsive frontend landing page built using **HTML, CSS, and JavaScript**.
The design includes a fullscreen hero slider, product listing grid, and responsive layout based on the provided Figma reference.

---

# Features

* Responsive Hero Slider
* Swiper.js Integration
* Autoplay & Infinite Loop
* Navigation Arrows
* Pagination Dots
* Responsive Product Grid
* Desktop / Tablet / Mobile Layout
* Clean UI Design
* Semantic HTML
* Responsive CSS
* JavaScript DOM Functionality

---

# Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Swiper.js
